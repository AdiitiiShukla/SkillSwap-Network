# SkillSwap-Network

## Problem Statement
Many students have valuable skills but lack access to affordable learning opportunities. Paid courses and certifications create barriers to learning.

## Solution
SkillSwap Network is a barter-based skill exchange platform where students can exchange skills instead of money. Users can offer what they know and learn what they need.

## Key Features
- Skill Offering – Users list skills they can teach  
- Skill Requesting – Users request skills they want to learn  
- Peer-to-Peer Matching – Connects learners and mentors  
- Open Participation – No fees or subscriptions  
- Community-driven learning environment  

## Google Technologies Used
- Google Forms – Collect skill offers and requests  
- Google Sheets – Store user data  
- Google Charts / Looker Studio – Analyze skill trends  
*(Planned / Conceptual Integration)*

## Tech Stack
- Python  
- Streamlit  
- Replit  
- GitHub  

## User Flow
1. User visits SkillSwap Network  
2. Chooses to offer or learn a skill  
3. Submits details  
4. Data is stored  
5. Matching happens based on skills  
6. Users connect and exchange skills
